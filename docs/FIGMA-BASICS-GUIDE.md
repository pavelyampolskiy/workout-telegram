# Основы Figma — краткое руководство для проекта

## Что такое Figma

Figma — это браузерный инструмент для дизайна интерфейсов. Позволяет создавать макеты экранов, прототипы, дизайн-системы и работать над ними в команде в реальном времени.

Основные преимущества:
- Работает в браузере — не нужно ничего устанавливать (есть и десктоп-приложение)
- Совместная работа в реальном времени (как Google Docs)
- Бесплатный тариф для небольших команд
- Встроенное прототипирование и dev-mode для разработчиков

---

## Быстрый старт

### 1. Создание аккаунта

Зайдите на [figma.com](https://figma.com), создайте аккаунт. Бесплатного тарифа (Starter) достаточно для начала: до 3 файлов, неограниченные черновики.

### 2. Основные понятия

| Понятие | Описание |
|---------|----------|
| **File** | Файл дизайна. Содержит страницы с макетами |
| **Page** | Страница внутри файла. Например, «Screens», «Components», «Flows» |
| **Frame** | Контейнер для дизайна. Может представлять экран, секцию или компонент |
| **Component** | Переиспользуемый элемент (кнопка, карточка, иконка). Изменения в компоненте обновляют все его копии |
| **Instance** | Копия компонента. Можно менять текст/цвет, но структура наследуется от основного компонента |
| **Auto Layout** | Автоматическое выравнивание элементов внутри фрейма (аналог flexbox в CSS) |
| **Style / Variable** | Переменные для цветов, шрифтов, отступов — основа дизайн-системы |

### 3. Горячие клавиши (самые нужные)

| Клавиша | Действие |
|---------|----------|
| `F` | Создать Frame |
| `R` | Прямоугольник |
| `T` | Текст |
| `V` | Выбор (Move tool) |
| `Ctrl/Cmd + D` | Дублировать |
| `Ctrl/Cmd + G` | Сгруппировать |
| `Shift + A` | Добавить Auto Layout |
| `Ctrl/Cmd + Alt + K` | Создать компонент |
| `Ctrl/Cmd + 0` | Показать всё на холсте |
| `Ctrl/Cmd + 1` | Масштабировать к выделению |

## Работа с Figma для Telegram Mini App

### Настройка фрейма экрана

Для Telegram Mini App рекомендуемые размеры фрейма:
- **Ширина**: 375–390 px (стандартный мобильный)
- **Высота**: 680–812 px (без системного статус-бара Telegram)

Создайте фрейм (`F`), задайте размеры в правой панели.

### Дизайн-токены проекта

В нашем проекте уже есть `design-tokens.js` с цветами и стилями. При создании макетов в Figma используйте те же значения:

| Токен | Назначение |
|-------|------------|
| Основной фон | Тёмная тема (`#1a1a2e` или аналогичный) |
| Текст основной | `rgba(255,255,255,0.9)` |
| Текст вторичный | `rgba(255,255,255,0.5)` |
| Акцент | Цвет из проекта (кнопки, ссылки) |
| Карточки | Полупрозрачный фон с закруглёнными углами |

Для синхронизации Figma-токенов с кодом существует плагин **Tokens Studio** (бывший Figma Tokens).

### Структура файла для проекта

Рекомендуемая организация страниц в Figma-файле:

```
📄 Cover          — обложка файла
📄 Components     — все переиспользуемые компоненты (кнопки, карточки, инпуты)
📄 Screens        — готовые экраны приложения
📄 Flows          — прототипы (связи между экранами)
📄 Archive        — устаревшие макеты
```

## Создание компонентов

### Пример: кнопка

1. Нарисуйте прямоугольник (`R`) — скруглите углы (например, 12 px)
2. Добавьте текст (`T`) по центру
3. Выделите оба элемента, добавьте Auto Layout (`Shift + A`)
4. Задайте отступы (padding): например, 16 по горизонтали, 12 по вертикали
5. Выделите фрейм → `Ctrl/Cmd + Alt + K` — теперь это компонент
6. Добавьте варианты (Variants): Default, Hover, Disabled

### Пример: карточка тренировки

1. Создайте фрейм шириной 343 px (375 − отступы 16×2)
2. Auto Layout, vertical, gap 8
3. Внутри: заголовок (название дня), список упражнений, статистика
4. Скруглите углы (12–16 px), добавьте фон карточки
5. Превратите в компонент

## Прототипирование

1. Перейдите на вкладку **Prototype** (правая панель)
2. Выделите элемент (например, карточку дня)
3. Перетащите синий кружок на целевой фрейм (экран упражнений)
4. Настройте анимацию: **Smart Animate**, 300 ms, Ease In-Out
5. Нажмите ▶ Play для предпросмотра

## Dev Mode (для разработчиков)

Dev Mode — режим Figma для разработчиков. Позволяет:
- Просматривать CSS-свойства элементов
- Копировать значения цветов, размеров, отступов
- Видеть расстояния между элементами (зажмите `Alt` и наведите)
- Экспортировать ассеты (иконки, изображения)

Переключение: кнопка `</>` в верхней панели или `Shift + D`.

## Передача макетов в разработку

1. **Поделитесь файлом** — нажмите Share → скопируйте ссылку с правами «Can view»
2. **Dev Mode** — разработчик открывает файл и переключается в Dev Mode
3. **Inspect** — клик на любой элемент показывает CSS, размеры, цвета
4. **Export** — иконки экспортируйте как SVG, изображения как PNG/WebP

### Именование слоёв

Для удобства разработчиков называйте слои осмысленно:
- `btn/primary`, `btn/secondary` — кнопки
- `card/workout`, `card/exercise` — карточки
- `icon/calendar`, `icon/timer` — иконки
- `screen/home`, `screen/history` — экраны
